ハンズオン用サンプルデータ
==========================

研修中に「自分の業務データはまだ手元にない」「機密情報なので試しづらい」という場合のため、
架空企業「株式会社サンプル商事」のサンプルデータを4種類用意しました。

サンプル商事の設定
------------------
  業種：BtoB卸売（食品 / 日用品 / オフィス用品）
  社員数：約230名
  拠点：東京本社 / 大阪・名古屋・福岡支店
  事業部：食品事業部 / 日用品事業部 / オフィス用品事業部
  決算月：3月

ファイル一覧
------------

1. サンプル商事_業界レポート2026.pdf
     → Day 2 ハンズオン①「PDFを要約」で使う
     → 全10ページ、卸売業界の構造変化と提言を扱った架空レポート

2. サンプル商事_経営会議議事録_2026年4月.txt
     → Day 2 ハンズオン②「議事録 → ToDo」で使う
     → 30分相当の経営会議文字起こし。決定事項・ToDo・未決事項が抽出可能

3. サンプル商事_月次データ_2025-11_2026-04.csv
     → Day 3 ハンズオン①「/xlsx で月次レポート」で使う
     → 6ヶ月分・3事業部の月次データ。事業部別の傾向差（食品=成長、オフィス用品=縮小）が読み取れる

4. サンプル商事_来期戦略メモ.docx
     → Day 3 ハンズオン③「/pptx でスライド」の元ネタ文書として使う
     → 上記月次データを踏まえた戦略メモ（2ページ程度）

使い方
------
本研修では「自分の本物のデータで使うこと」を最終目的としています。
サンプルデータは、まず安全に練習したい時／データを忘れた時の保険としてお使いください。

注意事項
--------
本サンプルデータに登場する企業名・人名・数字・分析内容は、すべて研修用に作成された架空のものです。
実在の企業・人物・データとは一切関係ありません。
